# good-cookbook
成功且可复现的菜谱

# 馕
```
1. 配方
   中筋粉 300g
    盐 5g
    糖 5g
    酵母 3g
    牛奶 160g
2. 做法
    揉好醒 15 分钟，  整形后 醒 15分钟。  揉成 3个小馕 ， 或 1个大馕。表面涂抹蛋液，撒芝麻，扎馕眼， 醒15分钟。
    放入 200度 预热好的烤箱，烤20分钟。 
```

# 那不勒斯披萨
