# 馕
```
耗时： 65分钟
1. 配方
   中筋粉 300g
    盐 5g
    糖 5g
    酵母 3g
    牛奶 160g
2. 做法
    揉好醒 15 分钟，  整形后 醒 15分钟。  揉成 3个小馕 ， 或 1个大馕。表面涂抹蛋液，撒芝麻，扎馕眼， 醒15分钟。
    放入 200度 预热好的烤箱，烤20分钟。 
```

# 那不勒斯披萨
```
耗时 24小时+ 15分钟
1. 配方
    高筋粉 500g
    水    300g
     盐   15g
     酵母 1g
2. 做法
    揉好发酵24小时 ，
    烤箱 400度 烘烤 3 分钟 ，  或 200度 预热好的烤箱烘烤15分钟
     揉面手法见视频
 3. 意大利配菜： 厚涂一层番茄酱， 放几片罗勒， 几片西红柿， 出锅后立刻放奶酪
 4. 大学食堂版配菜： 厚涂一层蛋液， 放上剩菜，比如酸辣土豆丝， 出锅后立刻喷醋和盐水
```

# 皮塔饼
```
耗时： 14.5小时
1. 配方
   中筋粉（11g蛋白） 500g
   清水 400g
   酵母 7g  酵母用最烂的酵母， 好酵母会导致发酵速度过快
   玉米粉/粗粮粉/麸皮粉 一袋，5-10kg
2. 做法
  1. 面粉 200g ， 清水 200g， 1g 酵母， 均匀搅拌成面糊后，盖上保鲜膜发酵12到16小时。 发酵为原始体积3倍大
  2. 发酵完成后，放入厨师机，开启慢速搅拌几十秒后 加300g 面粉， 10g 盐，10g 糖，6g 酵母，
  3. 继续慢速搅拌，在1分钟内逐步加入100ml清水，
  4. 当完全搅拌均匀后，标准速度搅拌5分钟。
  5. 调整为慢速搅拌并倒入50ml清水， 搅拌5分钟后成为面糊
  6. 调整为标准速度并倒入50ml清水，搅拌5分钟后成为Q弹高含水面团
  7. 静置发酵 1到2小时， 发酵为原始体积2到3倍大
  8. 将  玉米粉/粗粮粉/麸皮粉 倒入敞口大盆中
  9. 按 https://youtu.be/d-sFn81WzWo?t=507 手法 揪出约100g面团 ， 扔到 大盆里。 将大盆盖好后醒大约45分钟到一个小时
```

#鸡蛋灌饼
```
耗时：1小时
1. 配方
  面：
   普通面粉 500g
   盐 5g
   糖 10g
  油酥：
   面粉 60g
   盐 3g
   十三香 1g
   熟油 40g

2. 做法
2.1 面
   1. 放完面粉，盐，白糖
   2. 放100g 刚烧开的开水烫面。 搅拌均匀。
   3. 放200g 常温水。搅拌均匀后开始揉面
   4. 揉好后盖盖醒5分钟
   5. 重新揉后盖盖醒5分钟
   6. 重新揉后盖盖醒10分钟
   7. 搓成长条，分成8个面剂
   8. 面喷油，横断面向上摆放后搓成长条
   9. 全部搓好后放盆中盖上盖子醒30分钟
2.2 油酥
   1. 将所有配料放在小碗中，搅拌至看不见干粉
2.3 做法见视频   
```
