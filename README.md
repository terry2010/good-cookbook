# 馕
```
1. 配方
   中筋粉 300g
    盐 5g
    糖 5g
    酵母 3g
    牛奶 160g
2. 做法
    揉好醒 15 分钟，  整形后 醒 15分钟。  揉成 3个小馕 ， 或 1个大馕。表面涂抹蛋液，撒芝麻，扎馕眼， 醒15分钟。
    放入 200度 预热好的烤箱，烤20分钟。 
```

# 那不勒斯披萨
```
1. 配方
    高筋粉 500g
    水    300g
     盐   15g
     酵母 1g
2. 做法
    揉好发酵24小时 ，
    烤箱 400度 烘烤 3 分钟 ，  或 200度 预热好的烤箱烘烤15分钟
     揉面手法见视频
 3. 意大利配菜： 厚涂一层番茄酱， 放几片罗勒， 几片西红柿， 出锅后立刻放奶酪
 4. 大学食堂版配菜： 厚涂一层蛋液， 放上剩菜，比如酸辣土豆丝， 出锅后立刻喷醋和盐水
```
来源： https://www.bilibili.com/video/BV11M4y1P7WN

#鸡蛋灌饼
```
1. 配方
  面：
   普通面粉 500g
   盐 5g
   糖 10g
  油酥：
   面粉 60g
   盐 3g
   十三香 1g
   熟油 40g

2. 做法
2.1 面
   1. 放完面粉，盐，白糖
   2. 放100g 刚烧开的开水烫面。 搅拌均匀。
   3. 放200g 常温水。搅拌均匀后开始揉面
   4. 揉好后盖盖醒5分钟
   5. 重新揉后盖盖醒5分钟
   6. 重新揉后盖盖醒10分钟
   7. 搓成长条，分成8个面剂
   8. 面喷油，横断面向上摆放后搓成长条
   9. 全部搓好后放盆中盖上盖子醒30分钟
2.2 油酥
   1. 将所有配料放在小碗中，搅拌至看不见干粉
2.3 做法见视频   https://www.bilibili.com/video/BV1s54yexE8F?t=107.0
```
